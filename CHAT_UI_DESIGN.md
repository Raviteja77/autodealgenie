## Color Scheme

- **User Messages**: Blue background (#1976d2), white text
- **AI Messages**: White background, dark text
- **Background**: Light grey (#f5f5f5) for message area
- **Accents**: Primary blue for icons and active elements
- **Avatars**: 
  - AI: Primary blue with robot icon
  - User: Secondary color with person icon

## Interactive Elements

### Actions Tab
- Three prominent buttons arranged horizontally
- **Accept**: Green color, checkmark icon
- **Counter**: Blue color, money icon
- **Reject**: Red color, cancel icon
- Buttons disabled based on context (e.g., Accept only when offer exists)

### Chat Tab
- Text input with multiline support
- Attach button (ðŸ“Ž) to toggle dealer info mode
- Send button (ðŸ“¤) becomes loading spinner when sending
- Character counter shows remaining characters

### Dealer Info Mode
- Dropdown for info type selection (Price Quote, Inspection Report, etc.)
- Optional price input field
- Cancel button to exit dealer info mode
- All inputs validated before sending

## Message Features

### Message Bubbles
- Rounded corners (8px border radius)
- Elevation/shadow for depth
- Maximum width: 75% of container
- Padding: 12px

### Message Metadata
- Timestamp in smaller, lighter text
- Round indicator (collapsed by default, can expand)
- Dealer info badge when applicable
- Suggested price display when present
- Recommended action chip when provided

### Round Grouping
- Collapsible sections by round number
